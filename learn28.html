第 28 課


1.
Verb ます form ----> ますဖြုတ် ---- > なからပေါင်း (___ရင်းနဲ့___)

eg : ごはんをたべなからテレビをみます。
: はたらきながらにほんごをべんきょうしています。

2.
Verb て form + います (____နေတယ်/ထားတယ်)
eg : にほんごをべんきょうしています。

3.
Verb Plain form ---->
い ( adj ) ------------------> + し ( တဲ့အပြင် )
Noun / な( adj ) + だ ---->

eg :このみせはしずかだし,ひろいし,いつもここでかいています。

4.
Verb + た form ----> たり ____たり
( လိုက်____လိုက် )

eg : きのうのぼんたべたりねたりしました。

5.
____ も ____ し ____も
( လည်းပဲ ___တဲ့အပြင် ____လည်းပဲ )

eg : ミラ…さんはㇳラプもするしおさけものみます。

6.
Noun + も + verb( plain ) + し + Noun + も
( အကြောင်းပြချက်ပေးတဲ့ နေရာမှာသုံး )

eg : どうしてにほんのアニナがすぎですか？
: にほんごもべんきょうできるし。
: にほんのこともわかりますから。
