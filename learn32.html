第 32 課


1.
Verb た / ない + ほうがいいです。-------> တာကကောင်းတယ်

eg : まいにち うんどうしたほうがいいです。
: おさけをのまないほうがいいです。

2.
Verb ( plain ) ------->
い( adj ) い --------->
な ( adj ) なဖြုတ် ------> + でしょう ( မဟုတ်လား )
Noun ------------->

eg : こんばん あめでしょう
: いかないでしょう
: きれいでしょう / あついでしょう / のむでしょう

3.
Verb ( plain ) ------->
い ( adj ) い--------->
な ( adj ) なဖြုတ် ------> + かもしれません ( ဖြစ်ကောင်းဖြစ်လိမ့်မယ် )

Noun ------------->

eg : あしたこない/くるかもしれません。
: あしたいそがしいかもしれません。
: らいしゅうひまかもしれません
: まだあわないかのじょうです からきれいかもしれません。
