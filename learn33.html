1.
めいれいけい ( ဆရာမှ ကျောင်းသား / မိဘမှသားသမီး / အထက်လူကြီးမှ လက်အောက်ငယ်သား )

Verb 1
い လိုင်းမှ え လိုင်းပြောင်း ますဖြုတ်

eg : いきます ------> いけ
: しにます -------> しね

Verb 2
ます ဖြုတ် ろ ပေါင်း

eg : たべます -----> たべろ
: ねます ------> ねろ
Verb 3
かきま -------> こい
します -------> しろ(べんきょうします -------> べんきょうしろ)

2.
きんしけい ( ____ မ ____နဲ့ ) -----> စိတ်ဆိုး ရန်ဖြစ် ချကြတဲ့အခါမှာသုံး

Verb (Dic) + な

eg : みます --------> みるな
: いきます -------> いくな
: たべます --------> たべるな
: はなします -------> はなあな

3.
Verb ます -------> ますဖြုတ် なさいပေါင်း ( _______ပါ ) ----> နူးညံ့စွာ ခိုင်းစေခြင်း

eg : たべます ------->たべなさい
: れます ------> ねなさい
: あしたしけんですからべんきょうしなさい

4.
Verb ( Dictionary ) ----->
い ( adj ) い ----------------->
な ( adj ) ----------------------->といういみです ( ___ဆိုတဲ့ အဓိပ္ပာယ်ပါ )
Noun ------------->
eg :[きけん]はあぶないといういみです。
: 「だいすき」はとてもききといういみです。

5.
Verb ( Dictionary ) + といっていました ( ____လို့ပြောခဲ့တယ် )

eg : ミンさんはりょうこうへいくといっていました。

6.
もう __ Verb ( ない ) -----> နောက်ထက် _____မ__နိုင်တော့ဘူး

eg : もう たべられない
: もうめぼれない
: もういけない/もういかない

Verb ( plain ) -------->

eg : せんせいに10ぷんほどおくれるとったえていただけませんか
