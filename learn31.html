第 31 課


1.
Verb 1 -----> い လိုင်းမှ お လိုင်းပြောင်း う ပေါင်း ます ဖြုတ်

eg : いきます ----> いこう ( いきましょう)
: かいます ----->カおうし ( かいましょう)

Verb 2 -----> ますဖြုတ် よう ပေါင်း

eg : たべます -----> たべよう( たべましょう )
: ねます -----> ねよう( ねましょう )

Verb 3 ------>
きます -----> こよう
します ------>しよう
べんきょうします ------>べんきょうしよう

2.
いこうけい + とおもっています。-----> ( လို့တွေးထားတယ် )

eg : こんばんカラオケへいこうとおもっています。
: くちょいをふんでなげようとおもっています。

3.
V ( dic ) / ない + つもりです。------> ဖို့ ရည်ရွယ်ထားတယ်

eg : いつか にほんへいくつもりです。
: いったら かえらないつもりです。

4.
Verb ( dictionary ) / Noun + の + よていです အစီစဉ်ရှိ/ ဖြစ် / ထား

eg : らいしゅう ひっこしするよていです。
: りょうこうのよていはあしたです。
