第 30 課


1.
Verb て あります ----->ထားတယ် ( နဂိုကတည်းကထားတယ် )

eg : かべにえがかけてあります。
: はくばんに あいしてるとかいてあります。
: かべにきょうしつのよていがかけてあります。

2.
V て + おきます ----->ထားမယ် ( အနာဂါတ် ) ကြိုတင်လုပ်ပြီးသား

eg : あしたしけんがあるのでべんきょうしておきます。
: こどもがうまれるまえに、ふくやベッドをかっておきます。
: いすはそのままにしておいてください/おきます。

3.
Verb て + おいてください -----> ထားပေးပါ

4.
Verb ( dictionary ) ----->
Noun + の --------> まえに ( ____မ____ခင် )

5.
また + Verb ています。 -----> နေတုန်း/နေဆဲ

eg : また、たべています。
: また、いっています。
